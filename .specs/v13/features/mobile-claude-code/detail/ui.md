# 화면 구성

## 타임라인 탭 (기본)

```
┌─────────────────────────────┐
│  ☰  project / claude   [⌨]  │  ← 네비게이션 바
│─────────────────────────────│
│  "버그 수정해줘" · 5분 전     │  ← 메타 바 (컴팩트, 클릭 → 바텀 시트)
│─────────────────────────────│
│                             │
│  14:30 사용자                │
│  ┌────────────────────────┐ │
│  │ 버그 수정해줘            │ │
│  └────────────────────────┘ │
│                             │
│  14:31 ✅ Edit main.ts      │  ← 타임라인 엔트리
│  ▸ diff 보기                │
│                             │
│  14:32 Claude               │
│  ┌────────────────────────┐ │
│  │ 수정 완료했습니다...     │ │
│  └────────────────────────┘ │
│                             │
│─────────────────────────────│
│  메시지를 입력하세요...   ⏎  │  ← 하단 고정 입력창
│─────────────────────────────│
│    safe-area-inset-bottom   │
└─────────────────────────────┘
```

## 터미널 탭

```
┌─────────────────────────────┐
│  ☰  project / claude   [💬] │  ← 토글: 타임라인으로 전환
│─────────────────────────────│
│                             │
│  $ claude --resume abc123   │
│  ╭─────────────────────╮    │  ← 전체 화면 터미널 (축소 없이 100%)
│  │ > _                 │    │
│  ╰─────────────────────╯    │
│                             │
│─────────────────────────────│
│  메시지를 입력하세요...   ⏎  │  ← 입력창 유지
│─────────────────────────────│
│    safe-area-inset-bottom   │
└─────────────────────────────┘
```

## 토글 버튼

| 현재 탭 | 버튼 아이콘 | 의미 |
|---|---|---|
| 타임라인 | `Terminal` (`size={18}`) | "터미널로 전환" |
| 터미널 | `MessageSquare` (`size={18}`) | "타임라인으로 전환" |

- 활성 상태: `text-foreground`
- 비활성: `text-muted-foreground`
- 전환 애니메이션: fade 100ms

## 하단 고정 입력창

- 데스크톱 WebInputBar 재활용
- `position: sticky`, `bottom: 0`
- `padding-bottom: env(safe-area-inset-bottom)`
- 최대 높이: 3줄 (모바일 제한, 데스크톱은 5줄)
- 3가지 모드 동일: 입력/중단/비활성

### 중단 모드 (모바일)

```
┌─────────────────────────────┐
│  Claude가 응답 중...     ■   │
└─────────────────────────────┘
```

- 중단 버튼 클릭 → AlertDialog (데스크톱과 동일)

## 메타 바 바텀 시트

컴팩트 메타 바 클릭 시 바텀 시트로 상세 표시:

```
┌─────────────────────────────┐
│                             │
│  ─── (drag handle) ───     │
│                             │
│  "버그 수정해줘"             │
│                             │
│  생성   03/21 14:30 (2시간 전)│
│  수정   03/21 16:45 (5분 전) │
│  메시지  사용자 12 / 어시 15  │
│  ⑂ feature/fix-bug          │
│  토큰   입력 50K / 출력 30K  │
│                             │
└─────────────────────────────┘
```

- shadcn/ui `Drawer` 또는 `Sheet` side="bottom"
- 드래그 핸들로 닫기 가능
- 외부 클릭으로 닫기
