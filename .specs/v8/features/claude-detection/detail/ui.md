# 화면 구성

## 개요

claude 명령어 감지는 클라이언트 xterm `onTitleChange` 이벤트 기반이므로 별도 UI 요소가 없다. 감지 결과에 따른 Panel 타입 자동 전환의 시각적 효과를 정의한다.

## 자동 전환 시 시각 효과

### Terminal → Claude Code 자동 전환

```
BEFORE (Terminal 모드):
┌──────────────────────────────────────────┐
│ [Tab1]                    [─] [│] [✕]    │
├──────────────────────────────────────────┤
│ $ claude                                 │
│ ╭────────────────────────────────────╮   │
│ │ > 버그 수정해줘                     │   │
│ ╰────────────────────────────────────╯   │
│                                          │
└──────────────────────────────────────────┘

    ↓ onTitleChange → isClaudeProcess → panelType 자동 전환

AFTER (Claude Code 모드):
┌──────────────────────────────────────────┐
│ [Tab1]                 [⬚] [─] [│] [✕]  │
├──────────────────────────────────────────┤
│  ┌─ 타임라인 ─────────────────────────┐  │
│  │ (세션 데이터 로드 중 → 스켈레톤)    │  │
│  │ ...                                │  │
│  └────────────────────────────────────┘  │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  ┌─ 터미널 (축소) ────────────────────┐  │
│  │ $ claude                           │  │
│  └────────────────────────────────────┘  │
└──────────────────────────────────────────┘
```

- 전환 애니메이션: 타임라인 영역이 상단에서 슬라이드 인 (150ms ease-out)
- 터미널 영역: 전체 → 축소로 부드럽게 전환
- 탭 바: Panel 전환 토글 아이콘 자동 추가

### claude 종료 후

- Panel 타입 `claude-code` 유지
- 타임라인은 마지막 상태 그대로 표시 (정적)
- 사용자가 수동으로 Terminal 모드로 전환 가능

## 감지 상태 표시

감지 동작은 xterm 이벤트 기반으로 백그라운드에서 이루어지며, 사용자에게 감지 상태를 별도 표시하지 않는다. 전환 결과만 시각적으로 반영된다.
