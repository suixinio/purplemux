# 화면 구성

## 1. 에이전트 목록 페이지 (`/agents`)

### 레이아웃

```
┌─────────────────────────────────────────────────┐
│  [앱 헤더]                                       │
├─────────────────────────────────────────────────┤
│                                                  │
│  My Agents                          [+ 새 에이전트] │
│                                                  │
│  ┌──────────────┐  ┌──────────────┐             │
│  │ 🟢 backend   │  │ ⚪ frontend  │             │
│  │ 백엔드 개발   │  │ 프론트 개발   │             │
│  │              │  │              │             │
│  │ project-a    │  │ project-b    │             │
│  │ project-b    │  │              │             │
│  └──────────────┘  └──────────────┘             │
│                                                  │
└─────────────────────────────────────────────────┘
```

### 페이지 헤더

| 요소 | 스타일 |
|------|--------|
| 제목 | `text-xl font-semibold` |
| 생성 버튼 | `Button variant="default" size="sm"` + `Plus` 아이콘 |
| 간격 | `flex justify-between items-center mb-6` |

### 에이전트 카드

```
┌────────────────────────────┐
│  [상태뱃지] 에이전트명       │  ← 헤더
│  역할 텍스트                │  ← 서브텍스트
│                            │
│  📁 project-a              │  ← 프로젝트 목록
│  📁 project-b              │
└────────────────────────────┘
```

| 요소 | 스타일 |
|------|--------|
| 카드 컨테이너 | `border rounded-lg p-4 cursor-pointer hover:border-foreground/30 transition-colors` |
| 상태 뱃지 | 아래 "상태 뱃지 상세" 참조 |
| 에이전트명 | `text-sm font-medium` |
| 역할 | `text-xs text-muted-foreground mt-0.5` |
| 프로젝트 목록 | `text-xs text-muted-foreground mt-3 space-y-1` |
| 프로젝트 아이콘 | `Folder` size=12 `text-muted-foreground/60` |
| 카드 그리드 | `grid grid-cols-2 gap-3` (모바일: `grid-cols-1`) |

### 상태 뱃지 상세

| 상태 | 색상 | 아이콘 | 애니메이션 |
|------|------|--------|-----------|
| `idle` | `bg-muted-foreground/20` | 원형 6px | 없음 |
| `working` | `bg-ui-teal` | 원형 6px | `animate-pulse` |
| `blocked` | `bg-ui-amber` | 원형 6px | `animate-pulse` |
| `offline` | `bg-muted-foreground/10` | 원형 6px | 없음, 투명도 낮음 |

## 2. 에이전트 생성 다이얼로그

```
┌─────────────────────────────────────┐
│  새 에이전트 만들기             [X]  │
├─────────────────────────────────────┤
│                                     │
│  이름                               │
│  ┌─────────────────────────────┐   │
│  │ backend-bot                  │   │
│  └─────────────────────────────┘   │
│                                     │
│  역할                               │
│  ┌─────────────────────────────┐   │
│  │ 백엔드 개발 전담              │   │
│  └─────────────────────────────┘   │
│                                     │
│  담당 프로젝트                       │
│  ┌─────────────────────────────┐   │
│  │ ☑ project-a                  │   │
│  │ ☑ project-b                  │   │
│  │ ☐ project-c                  │   │
│  └─────────────────────────────┘   │
│                                     │
│             [취소]  [만들기]          │
└─────────────────────────────────────┘
```

### 폼 필드

| 필드명 | 컴포넌트 | 필수 | 검증 규칙 |
|--------|----------|------|-----------|
| 이름 | `Input` | Y | 1~30자, 영문+숫자+하이픈, 중복 불가 |
| 역할 | `Input` | N | 최대 100자 |
| 담당 프로젝트 | 체크박스 리스트 | N | 기존 워크스페이스 목록에서 선택 |

| 요소 | 스타일 |
|------|--------|
| 다이얼로그 | `Dialog` (shadcn) `max-w-md` |
| 라벨 | `text-sm font-medium mb-1.5` |
| 입력 필드 간격 | `space-y-4` |
| 프로젝트 체크박스 | `Checkbox` (shadcn) + `text-sm` |
| 버튼 영역 | `flex justify-end gap-2 mt-6` |
| 만들기 버튼 | `Button variant="default"` — 생성 중 `Loader2 animate-spin` |

## 3. 에이전트 설정 시트

카드 우측 상단 `Settings` 아이콘 클릭 시 `Sheet` (shadcn) side="right".

- 생성 다이얼로그와 동일한 폼 필드
- 하단에 삭제 버튼: `Button variant="destructive" size="sm"` — "에이전트 삭제"

## 4. 삭제 확인 다이얼로그

```
┌─────────────────────────────────────┐
│  에이전트 삭제                  [X]  │
├─────────────────────────────────────┤
│                                     │
│  "backend-bot"을 삭제하면            │
│  채팅 이력과 메모리가 모두            │
│  삭제됩니다.                         │
│                                     │
│  에이전트 이름을 입력하세요           │
│  ┌─────────────────────────────┐   │
│  │                              │   │
│  └─────────────────────────────┘   │
│                                     │
│             [취소]  [삭제]            │
└─────────────────────────────────────┘
```

- 삭제 버튼: 이름 입력이 정확히 일치할 때만 활성화
- `Button variant="destructive"` — 삭제 중 `Loader2 animate-spin`

## 5. 헤더 진입점

```
┌─────────────────────────────────────────────────┐
│  purplemux   [터미널] [노트] [통계] [🤖 ²]  ... │
└─────────────────────────────────────────────────┘
                                      ↑
                               에이전트 아이콘
                               blocked 카운트 뱃지
```

| 요소 | 스타일 |
|------|--------|
| 아이콘 | `Bot` (lucide) size=16 |
| 뱃지 | blocked 에이전트 수 > 0일 때 표시, `bg-ui-amber text-white text-[10px] rounded-full min-w-[16px] h-4` |
| 클릭 | `/agents` 페이지로 이동 |

## 로딩/빈/에러 상태

### 목록 로딩

```
┌──────────────┐  ┌──────────────┐
│  ████████    │  │  ████████    │
│  ██████      │  │  ██████      │
│  ████        │  │  ████        │
└──────────────┘  └──────────────┘
```

스켈레톤 카드 2개 표시 — `Skeleton` (shadcn) 활용

### 빈 상태

```
┌─────────────────────────────────────┐
│                                     │
│         🤖                          │
│   아직 에이전트가 없습니다            │
│   첫 에이전트를 만들어보세요          │
│                                     │
│       [+ 새 에이전트 만들기]          │
│                                     │
└─────────────────────────────────────┘
```

| 요소 | 스타일 |
|------|--------|
| 아이콘 | `Bot` size=32 `text-muted-foreground/40` |
| 텍스트 | `text-sm text-muted-foreground text-center` |
| CTA 버튼 | `Button variant="outline" size="sm"` |

### API 에러

- `toast.error('에이전트 목록을 불러올 수 없습니다')` + 재시도 버튼

## 접근성

| 항목 | 처리 |
|------|------|
| 카드 그리드 | `role="list"` |
| 각 카드 | `role="listitem"` `tabIndex={0}` — Enter로 채팅 이동 |
| 상태 뱃지 | `aria-label="상태: working"` |
| 생성 다이얼로그 | Dialog 기본 접근성 (shadcn) |
| 삭제 확인 | `aria-live="assertive"` |
