# 메모리 뷰어

## 개요

에이전트의 파일 기반 메모리(`~/.purplemux/agents/`)를 열람하고 편집하는 UI. 공유 메모리와 에이전트별 메모리를 계층적으로 탐색하며, 에이전트가 학습한 내용을 사용자가 검증하고 수정할 수 있다.

## 주요 기능

### 메모리 트리 탐색

- `~/.purplemux/agents/` 디렉토리 구조를 트리 뷰로 표시
  - `shared/` — 전체 공유 메모리
    - `user.md`, `global-learnings.md`
    - `projects/{project}/` — 프로젝트별
  - `{agent}/` — 에이전트별 메모리
    - `config.md`, `memory/`, `missions/`
- 현재 에이전트의 메모리를 기본 펼침, 나머지는 접힘
- 파일 아이콘으로 타입 구분 (설정, 학습, 미션 등)

### 파일 내용 뷰어

- 트리에서 파일 클릭 시 우측 패널에 마크다운 렌더링
- 파일 메타 정보: 크기, 최종 수정일, 수정한 에이전트
- 긴 파일은 접기/펼치기 지원

### 인라인 편집

- "편집" 버튼으로 마크다운 에디터 전환
- 실시간 미리보기 (split view)
- 저장 시 파일 직접 수정 — 에이전트가 다음 참조 시 반영됨
- 편집 이력은 별도 관리하지 않음 (파일 시스템 기반)

### 메모리 검색

- 전체 메모리 파일에서 키워드 검색
- 검색 결과: 파일명 + 매칭 라인 미리보기
- 검색 결과 클릭 시 해당 파일/위치로 이동

### 메모리 통계

- 전체 메모리 크기, 파일 수
- 에이전트별 메모리 사용량
- 최근 변경된 메모리 파일 목록

### 로딩/빈/에러 상태

- 로딩: 트리 + 패널 스켈레톤
- 빈 상태 (메모리 없음): "에이전트가 아직 학습한 내용이 없습니다"
- 파일 읽기 에러: 에러 메시지 + 파일 경로 표시
