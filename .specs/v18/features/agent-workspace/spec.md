# 에이전트 워크스페이스

## 개요

에이전트가 계획을 수립하고 태스크를 관리하는 전용 워크스페이스. 에이전트가 생성한 탭들을 사용자가 실시간으로 관찰할 수 있으며, 기존 purplemux 워크스페이스/탭 구조를 그대로 활용한다.

## 주요 기능

### 에이전트 전용 워크스페이스

- 에이전트 생성 시 자동으로 전용 워크스페이스 생성
- 에이전트의 "두뇌" 세션 (계획 수립, 판단) 탭이 기본으로 존재
- 에이전트가 태스크 실행을 위해 생성한 탭들이 자동으로 추가

### 탭 관찰

- 에이전트가 프로젝트 워크스페이스에 생성한 탭 목록 표시
- 각 탭: 이름, 연결된 Task, 현재 상태(실행 중/idle/완료)
- 탭 클릭 시 해당 터미널 탭으로 이동 — 에이전트가 Claude Code와 대화하는 과정을 실시간 관찰
- 관찰 모드: 사용자는 보기만 가능, 직접 입력은 차단 (실수로 에이전트 세션에 타이핑 방지)

### 워크스페이스 간 탭 분포

- 에이전트가 여러 프로젝트에 탭을 생성한 경우 프로젝트별로 그룹화 표시
- 각 프로젝트 그룹: 워크스페이스명 + 탭 목록
- 프로젝트 워크스페이스 클릭 시 해당 워크스페이스로 이동

### 에이전트 활동 요약

- 현재 실행 중인 태스크 수, 완료된 태스크 수
- 에이전트 가동 시간
- 최근 활동 타임라인 (간략한 로그)

### 로딩/빈/에러 상태

- 로딩: 워크스페이스 스켈레톤
- 빈 상태: "에이전트가 아직 작업을 시작하지 않았습니다"
- 에이전트 offline: "에이전트가 오프라인입니다" + 재시작 버튼
