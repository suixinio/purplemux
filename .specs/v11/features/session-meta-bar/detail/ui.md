# 화면 구성

## 전체 레이아웃

### 메타 바 위치

```
┌─ Claude Code Panel ──────────────────────┐
│  ┌─ 네비게이션 ────────────────────────┐  │
│  │  ← 세션 목록                        │  │
│  └────────────────────────────────────┘  │
│  ┌─ 메타 바 (컴팩트) ─────────────────┐  │  ← 고정, 스크롤 밖
│  │  "버그 수정해줘" · 5분 전 · 12턴 · 80K │  │
│  └────────────────────────────────────┘  │
│  ┌─ 타임라인 영역 ────────────────────┐  │  ← 스크롤 영역
│  │  ...                                │  │
│  └────────────────────────────────────┘  │
│  ┌─ 입력창 ──────────────────────────┐  │
│  │  메시지를 입력하세요...            │  │
│  └────────────────────────────────────┘  │
│  ┌─ 터미널 영역 ──────────────────────┐  │
│  │  $ claude --resume ...              │  │
│  └────────────────────────────────────┘  │
└──────────────────────────────────────────┘
```

## 컴팩트 레이아웃 (기본)

```
┌──────────────────────────────────────────────┐
│  "버그 수정해줘"  ·  5분 전  ·  12턴  ·  80.2K  │
└──────────────────────────────────────────────┘
```

- 전체: `px-4 py-1.5 cursor-pointer`, `border-b` (0.5px)
- 호버: `bg-muted/30`
- 클릭: 상세로 확장

| 항목 | 스타일 | 예시 |
|---|---|---|
| 제목 | `text-sm font-medium text-foreground truncate max-w-[200px]` | "버그 수정해줘" |
| 구분자 | `text-muted-foreground/50 mx-1.5` | · |
| 시간 | `text-xs text-muted-foreground` | 5분 전 |
| 턴 수 | `text-xs text-muted-foreground` | 12턴 |
| 토큰 | `text-xs text-muted-foreground font-mono` | 80.2K |

- 시간: Tooltip에 절대 시간 (`YYYY-MM-DD HH:mm:ss`)
- 토큰 포맷: 1,000+ → "1.2K", 1,000,000+ → "1.2M"

## 상세 레이아웃 (확장)

```
┌──────────────────────────────────────────────┐
│  "버그 수정해줘"                          ▾   │
│                                              │
│  생성     03/21 14:30 (2시간 전)              │
│  수정     03/21 16:45 (5분 전)                │
│  메시지   사용자 12 / 어시스턴트 15            │
│  ⑂ feature/fix-bug                           │
│  토큰     입력 50,234 / 출력 30,128 / 총 80,362│
└──────────────────────────────────────────────┘
```

- 전체: `px-4 py-3`, `bg-muted/30`, `border-b` (0.5px)
- 제목: `text-sm font-medium text-foreground` (전체 텍스트, 여러 줄 가능)
- 접기 아이콘: `ChevronDown` → `ChevronUp` (우측 상단)
- 확장 애니메이션: `transition-all duration-150 ease-out`

### 상세 필드

| 레이블 | 값 스타일 | 비고 |
|---|---|---|
| 생성 | `text-xs text-muted-foreground` | 절대 + (상대) |
| 수정 | `text-xs text-muted-foreground` | 절대 + (상대) |
| 메시지 | `text-xs text-muted-foreground` | 사용자 N / 어시스턴트 N |
| 브랜치 | `text-xs font-mono text-muted-foreground` | `GitBranch` 아이콘 (`size={12}`) + 이름 |
| 토큰 | `text-xs font-mono text-muted-foreground` | 입력 N / 출력 N / 총 N (쉼표 포맷) |

- 레이블: `text-xs text-muted-foreground/70 w-12` (고정 폭 정렬)
- 값: 레이블 우측에 인라인 배치
- 행 간격: `gap-1`

### 브랜치 로딩 상태

- 초기 로드 중: `text-xs text-muted-foreground/50` "로드 중..."
- git 저장소 아님: 브랜치 행 숨김
- 브랜치 조회 실패: 브랜치 행 숨김 (에러 표시 안 함)

## 숨김 상태

- 세션 목록 뷰 또는 `panelType === 'terminal'`: 메타 바 영역 `height: 0`, `overflow: hidden`
- 전환: `transition-all duration-150`

## 반응형/접근성

- 컴팩트 바: `role="button"`, `tabIndex={0}`, Enter/Space로 확장
- 상세 모드: Escape로 접기
- 외부 클릭: 상세 모드에서 접기
- 시간 Tooltip: 키보드 포커스 시에도 표시
